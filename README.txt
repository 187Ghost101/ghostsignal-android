GhostSignal IMSI Detector
=========================
IMPORTANT: Upload ALL these files to the ROOT of your GitHub repo (not in a subfolder!)

1. Create a public GitHub repo (github.com)
2. Upload ALL these files directly to the repo root
3. Go to Actions tab -> wait for build (~3 min)
4. Download APK from Artifacts
5. Install on Android (allow unknown sources)

API Endpoint: https://api.base44.com/api/apps/69bbda5b75a19519f5fc1e19/functions/receiveScan
